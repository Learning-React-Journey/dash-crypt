## DashCrypt

- Developed a secure authentication system using Login, Signup, Access & Refresh Tokens.
- Built a protected Dashboard with React Router and Context API for state management.
- Implemented CRUD operations for managing users: create, read, update, delete.
- Integrated Axios for API calls and error handling for forms.
- Ensured token-based session persistence using universal-cookie.
- Practiced Hooks (useState, useEffect, useContext), Protected Routes, and Conditional Rendering.
- Implemented Logout functionality and form validation with user-friendly error messages.
